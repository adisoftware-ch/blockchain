Blockchain Demo The Demo Simple Blockchain

Blockchain is - simply spoken - a database where all data is held by the users (Nodes).
- Trust concerning integrity of the data is given by irreversably sealing blocks of transactions (Mining).
- Completeness over the data is given by regularly negotiating about current complete and correct version (Consensus)


This Demo is intended to showcase these basic steps of a Blockchain:
- Start the App at least twice
- Have a look at the 'Node's
- Mine the first blocks; this activates the client's wallets
- Do some transactions between the clients
- Mine as you want; the blockchain grows

- Start a third (or fourth ...) App
- Look at it's Node - it's empty!
- Run 'Consensus' - it receives the chain!
- Go on playing arount ...


Note: The chain lives only in memory. It lives exactly as long as - worldwide - at least one Node (App) is alive!
