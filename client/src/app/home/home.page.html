Blockchain Demo The Demo Simple Blockchain

A Blockchain is - simply spoken - a database laking any central instance. All data is hold by distributed nodes. Trust over the data contained is given by sealing past Transactions and finding consensus between the Nodes. A malicious Node would require 51% of calculation power to break consensus.


This Demo is intended to showcase the basic steps of a Blockchain. How does it work:

  1. Client sends Transaction to the System
  2. All Nodes pick it up
  3. Other Transactions to follow, till ..
  4. At least one Node mines the next block of Transactions
  5. The mining Node notifies the others of the newly mined Block
  6. All Nodes pick the new Block; check it for validity (Proof of Work) and update their Blockchain
  7. Once accepted, the new Block is sealed; e.g. immutable without forking the Blockchain

Play around! Start the app in multiple browsers and multiple devices. Create Transactions as a Client and mine Blocks as a mining Node. Check the Log for messages sent and consumed by the distributed System.
